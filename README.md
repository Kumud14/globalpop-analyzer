# GlobalPop Analyzer

A data-driven project that tracks real-time global population statistics and visualizes insights using Power BI. The system reads live population data from Worldometer and allows analysis using dynamic dashboards.

##  Project Overview

- Automates fetching of world population data (births, deaths, net population).
- Stores and updates data in structured files.
- Visualizes trends using Power BI dashboards.
- Includes historical datasets for comparative analysis.

##  Tech Stack

| Component | Technology |
|----------|------------|
| Programming | Python  |
| Visualization | Power BI |
| Data Source | Worldometer + CSV datasets |
| Automation | Task Scheduler (Windows) |
